# Jedd's Tree Care Website

## Project Overview
This repository contains a website project completed as part of my coursework while pursuing a Bachelor of Business Administration in Information Systems and Technology (IST).

The project demonstrates fundamental web development concepts using HTML and CSS, including page structure, navigation, layout design, styling, and image integration.

## Technologies Used
* HTML5
* CSS3

## Files
* `jtc_index.html` – Home page
* `jtc_services.html` – Services page
* `jtc_base.css` – Base styles
* `jtc_layout.css` – Layout styles
* Image assets (`.png` files)

## Skills Demonstrated
* Website development
* HTML document structure
* CSS styling and layout
* Multi-page website navigation
* File and project organization

## Academic Background
Completed as part of coursework for the Bachelor of Business Administration in Information Systems and Technology program at Cleveland State University.
